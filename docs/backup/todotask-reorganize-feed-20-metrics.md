# 맛 표현 20종 중심 홈 피드 재구성 계획 (완료)

본 계획은 `planning_refine_taste_expressions.md`에서 정의한 20가지 미식 표준 지표를 홈 피드의 최우선 정보로 재배치하여, "숫자와 데이터만으로 맛을 완벽히 유추한다"는 야미덱스의 비전을 강화하는 것을 목표로 합니다.

## 1. 20대 미식 지표 체계화 (beta/script.js)
- [x] **지표 누락 확인 및 보강**: 
    - 1. 기본 미각(5): 단맛, 짠맛, 신맛, 감칠맛, 쓴맛
    - 2. 물리적 식감(7): 매운맛, 바삭함, 쫄깃함, 부드러움, 꾸덕함, 기름짐, 신선도
    - 3. 직관적 풍미(8): 치즈/우유, 버터향, 불맛, 향신료, 고소함, 육향, 해물향, 과일/상큼
- [x] **샘플 데이터 확장**: 각 피드당 최소 5~8개의 주요 지표를 포함하도록 데이터 셋 리뉴얼.

## 2. 홈 피드 UI 레이아웃 개편 (beta/index.html)
- [x] **지표 중심 섹션 배치**: 사진 바로 아래에 해당 음식의 '맛 성분표(20대 지표 요약)'를 가장 넓은 면적으로 배치.
- [x] **카테고리별 시각화**: 
    - 미각/식감/풍미 3대 카테고리를 색상 코드로 구분하여 사용자가 직관적으로 정보를 인지하도록 개선.
    - 단순히 수치 바(Bar)만 보여주는 것이 아니라, 카테고리별 그룹화를 통해 '맛의 구조'를 시각화.
- [x] **듀얼 바(나 vs 대중) 유지 및 강화**: 모든 핵심 지표에 대해 개인의 감각과 대중의 보편적 기준을 동시에 노출.

## 3. 스타일 및 인터랙션 고도화 (beta/style.css)
- [x] **20종 지표 전용 아이콘 시스템**: 각 지표별 이모지를 명확히 매핑하고 가독성 높은 폰트 및 컬러 적용.
