# 홈 피드 미식 데이터 시각화 고도화 계획 (완료)

본 계획은 `planning-upgrade-home-taste-data.md` 기획안을 바탕으로, 단순한 개인 기록을 넘어 '개인의 미각'과 '대중의 빅데이터'를 결합한 야미덱스만의 독창적인 피드 UI를 구축하는 것을 목표로 합니다.

## 1. 홈 피드 UI 구조 재편 (beta/index.html)
- [x] **[개인 vs 대중] 지표 비교 UI 도입**: 
    - 작성자의 수치와 대중 평균 수치를 동시에 보여주는 듀얼 바(Dual Bar) 시스템 구축.
    - 예: 작성자 바(진한 색) 위에 대중 평균 포인트(구분선 또는 연한 색) 표시.
- [x] **공식 미식 스펙트럼 섹션 추가**: 
    - 기존 '다른 시선 요약(댓글 분석)' 영역을 삭제.
    - 수천 명의 데이터를 종합한 해당 메뉴의 '상위 3개 핵심 맛 지표'를 시각적으로 강조.
- [x] **데이터 순도 정화**: 
    - 0~100 수치 지표에서 '양 적음', '와인페어링' 등 비미각 요소를 제거.
    - 해당 정보들은 하단에 별도 태그(#가성비 등)로 분리.

## 2. 스타일링 및 시각화 고도화 (beta/style.css)
- [x] **듀얼 바(Dual Bar) 스타일 정의**: 
    - 작성자의 개별적 감각(개성)과 대중의 보편적 기준(객관성)이 직관적으로 대비되도록 디자인.
- [x] **카테고리별 컬러 시스템 강화**: 기본 미각, 식감, 풍미 카테고리가 시각적으로 명확히 구분되도록 보정.
