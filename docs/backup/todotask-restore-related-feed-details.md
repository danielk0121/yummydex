## 1. '같은 메뉴 다른 시선' 섹션 UI 복구 (beta/index.html)
- [x] **유저별 핵심 지표 매핑**: 단순히 유저 아이디와 사진만 보여주는 것이 아니라, 해당 유저가 이 메뉴에 대해 느낀 핵심 지표(예: "매운맛 90!")를 함께 노출.
- [x] **시각적 일관성**: 개편된 20대 지표 스타일과 어울리도록 미니 카드 형태로 디자인 복구.
