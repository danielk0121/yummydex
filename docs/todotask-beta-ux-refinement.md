# 베타(Beta) UX/UI 개선 작업 계획

본 문서는 `uxanalysis-20260314-152000.md`와 `planning_refine_taste_expressions.md`를 기반으로 베타 버전의 완성도를 높이기 위한 작업 계획을 담고 있습니다.

## 1. UI 가독성 및 디자인 정밀 보정
- [x] **텍스트 대비 강화**: 피드 카드 내 시간 정보, 위치 정보 등의 연회색(`#8e8e93`)을 `#636366`으로 변경하여 가독성 확보.
- [x] **탭 타겟 최적화**: 탐색 화면 필터 칩 간격을 8px 이상으로 조정하고, 하단 네비게이션 클릭 영역 보정.
- [x] **브랜드 일관성**: 이모지 아이콘 사용 규칙 정리 및 UI 내 폰트 무게감(Weight) 재조정.

## 2. 사용자 경험(UX) 흐름 개선
- [x] **홈 온보딩 가이드**: 홈 화면 최상단에 "야미덱스가 처음이신가요? [가이드 보기]" 팝업 또는 배너 추가.
- [x] **검색 결과 없음(Empty State)**: 탐색 화면에서 검색 결과가 없을 때 일러스트(이모지)와 함께 "다른 맛을 찾아볼까요?" 문구 및 초기화 버튼 노출.
- [x] **마킹 단계 보완**: 맛 표현 단계에서 '좋은 예시'와 '나쁜 예시'를 텍스트나 툴팁으로 보강.

## 3. 데이터 및 기능 고도화 (20대 지표 중심)
- [x] **샘플 데이터 보강**: `beta/script.js`의 샘플 피드 데이터를 20대 지표(기본 미각, 식감, 풍미)가 골고루 섞이도록 리뉴얼.
- [x] **지표 시각화 디테일**: 막대 그래프(Bar) 외에 지표별 카테고리 아이콘 강조 및 컬러 코딩 정교화.
